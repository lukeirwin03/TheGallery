Welcome to the Gallery

The Gallery is a passion project that I made to showcase my programming abilities. The Gallery itself is a game made using Kaboom JS with all original artwork and sprites done by myself. Additionally, the character that you will be using in the game is my cat, Marcy.

The controls are pretty standard, you can use either WASD or the arrow keys to navigate. Please explore the map and check out the Gallery building to see my projects that I am showcasing. The source code is linked if you click on the icons on the desktop.

Please take a look around! I hope you get as much joy exploring around as I got from making this.

Also check out me out on GitHub and LinkedIn!
